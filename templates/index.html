Prediction of Balinese Vowel Characters Using Convolutional Neural Networks (CNN)

Pre Processing Data

Processes raw data into 1D and 3D features ready for model training through filtering, feature extraction, and spatial mapping.

Training Model

Train the model using the preprocessed EEG data to develop a predictive model for analysis and tasks.

Prediction

Prediction involves using the trained model to make forecasts of unseen EEG data based on learned patterns.
